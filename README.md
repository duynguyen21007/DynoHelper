# DynoHelper (Autonomous Robotics Project) | C++, Arduino UNO, L293D, HC-SR04

Programmed hardware-level control logic in C++ on an Arduino UNO to autonomously navigate a multi-obstacle course, managing a 3-motor drivetrain with a dual front-wheel configuration.

Implemented real-time spatial data processing using an HC-SR04 ultrasonic sensor to trigger precision braking within a 5cm threshold of target obstacles.

Engineered an automated payload delivery sequence by integrating an SG90 servo motor, coordinating timing and actuation thresholds to successfully launch a projectile over a 30cm barrier.
